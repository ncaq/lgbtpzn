LGBTPZNとは

LGBTPZNは、かつては、7種類の多様な性――レズビアン、ゲイ、バイセクシャル、トランスジェンダー、ペドフィリア、ズーフィリア、ネクロフィリア――のアクロニムでした。現在は、LGBTPZNはアクロニムではなく、多様な性を象徴するシンボルであると考えられています。

LGBTPZN.orgとは

LGBTPZN.orgは、自分自身をLGBTPZNであると感じている人と、その支援者による運動体です。この運動体は完全な非営利であり、また、意思決定のための中央集権的な機関を持っていません。LGBTPZN.orgはTwitterの #LGBTPZN ハッシュタグおよび #LGBTPZNさんと繋がりたい ハッシュタグでのラフ・コンセンサス (緩い総意) のもとで運営されます。 (補足: 運動体の成熟とともに、意思決定のためのルールと機関が整備される可能性があります。)

インターネットのドメイン名lgbtpzn.orgの所有者、および、LGBTPZN.orgのウェブサイト (いまあなたが見ているものです) の管理者は、墓場人夜 または @vaginaplant を自称しています。この管理者はLGBTPZNの運動体の指導者ではありません。墓場人夜は 不買運動ウェブ の作者でもあります。

LGBTPZN.org Wiki は誰でも編集できる自由なWikiです。編集には GitHub のアカウントが必要ですが、これは無料です。また、GitHubのアカウントを持っていれば、LGBTPZN.orgのウェブサイトにプルリクエストを送ることができます。

海外のLGBTPZN

ポーランドでのLGBTPZNは、保守的なカトリック支持者が、LGBTをおとしめるために造語したと言われています。これは、わが国でのLGBTPZNとは正反対であるように思えます。

しかしながら、LGBTPZNは多様な性を包摂するシンボルであり、その8番目のアルファベットは、ホモフォビアに対してすら開かれています。ポーランドのLGBTPZNと、わが国のLGBTPZNは、本質的には連帯可能であると考えられています。
